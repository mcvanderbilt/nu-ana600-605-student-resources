Only R packages that are allowed may be used in course materials. Use of unapproved packages may result in a grade of 0 or a decrease in points for an assignment. If identified, they will be removed before the code is executed for grading.

Loading of unused / unnecessary libraries will result in point deductions for assignments. You must evaluate your code to determine what packages are necessary.

Three methods of installing and loading approved R packages are provided at three different skill levels.

  • Level 1 - Individual script lines
  • Level 2 - Use of a looped function
  • Level 3 - Use of a standalone function
