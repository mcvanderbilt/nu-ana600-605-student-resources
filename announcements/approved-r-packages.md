# Installation & Loading of Approved R Packages
Only R packages that are allowed may be used in course materials. Use of unapproved packages may result in a grade of 0 or a decrease in points for an assignment. If identified, they will be removed before the code is executed for grading. Loading of unused / unnecessary libraries will result in point deductions for assignments. You must evaluate your code to determine what packages are necessary.

## Three methods of installing and loading approved R packages are provided at three different skill levels.
- [Level 1 - Individual Script Lines](https://github.com/mcvanderbilt/nu-ana600-605-student-resources/blob/main/scripts/allowed-packages_level-1.r)
- [Level 2 - Use of a Looped Function](https://github.com/mcvanderbilt/nu-ana600-605-student-resources/blob/main/scripts/allowed-packages_level-2.r)
- [Level 3 - Use of a Standalone Function](https://github.com/mcvanderbilt/nu-ana600-605-student-resources/blob/main/scripts/allowed-packages_level-3.r)
